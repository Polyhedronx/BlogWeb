## 关于这个博客

这个博客网站使用 **React + Go + PostgreSQL** 构建，以 Docker Compose 部署。

所有文章均以 Markdown 格式存储在 `content/posts/` 目录下，通过 Git 进行版本管理。

博客支持：
- 全文搜索
- 评论互动（支持嵌套回复和点赞）
- 暗色模式
- RSS 订阅
- LaTeX 数学公式渲染
