## 1. 对 Read View 四个字段的思考

- `m_ids` ：指的是在创建 Read View 时，当前数据库中「活跃事务」的事务 id 列表，注意是一个列表，“活跃事务”指的就是，启动了但还没提交的事务，并且不包括创建该 Read View 的事务自身。 
- `min_trx_id` ：指的是在创建 Read View 时，当前数据库中「活跃事务」中事务 id 最小的事务，也就是 `m_ids` 的最小值。 
- `max_trx_id` ：这个并不是 `m_ids` 的最大值，而是创建 Read View 时，下一个将要分配给新事务的事务 id 值（即全局事务 id 计数器的当前值）。 
- `creator_trx_id` ：指的是创建该 Read View 的事务的事务 id。

这里我有一个疑问：为什么要有 `max_trx_id`？明明用 `creator_trx_id` 就行了，只用改一下边界判断条件，还少一个字段。

求证后我发现这个问题的核心在于我搞错了一件事，Read View 并不是在创建事务时就分配了事务 id，而是在**第一次执行对数据库的更改操作时**才分配事务 id。比如如下这个场景：

1. 事务 A 启动，执行了一条 UPDATE，分配到 `trx_id = 10`（creator_trx_id 固定为 10）；
2. 接下来的 1 小时里，系统跑了 100 万个事务，全部正常提交，全局事务 ID 计数器涨到了 1000010；
3. 此时事务 A 执行第一条 SELECT，生成 Read View：`max_trx_id = 1000010`，而 `creator_trx_id = 10`。

如果用 `creator_trx_id` 当水位线，会错误地把 ID 11~1000009 这些早就提交的事务全部判定为不可见，完全不符合一致性视图的语义。

## 2.数学推导：为什么样本方差分母必须是 $n-1$

设总体均值为 $\mu$，总体方差为 $\sigma^2$。从总体中随机抽取 $n$ 个独立样本 $X_1, X_2, ..., X_n$，样本均值为 $\bar{X} = \frac{1}{n}\sum_{i=1}^n X_i$。

我们考察离差平方和 $S = \sum_{i=1}^n (X_i - \bar{X})^2$ 的数学期望，看它是否等于 $n\sigma^2$。

### 第一步：展开平方和

将平方和展开为单个平方项与均值平方项的组合：

$$
\begin{aligned}
E(S) 
&= E\left[\sum_{i=1}^n (X_i - \bar{X})^2\right] \\
&= E\left[\sum_{i=1}^n (X_i^2 - 2X_i\bar{X} + \bar{X}^2)\right] \\
&= E\left[\sum_{i=1}^n X_i^2 - 2\bar{X}\sum_{i=1}^n X_i + n\bar{X}^2\right]
\end{aligned}
$$

由于 $\sum_{i=1}^n X_i = n\bar{X}$，代入得：

$$
E(S) = E\left[\sum_{i=1}^n X_i^2 - 2n\bar{X}^2 + n\bar{X}^2\right] = E\left[\sum_{i=1}^n X_i^2 - n\bar{X}^2\right]
$$

即：

$$
E(S) = \sum_{i=1}^n E(X_i^2) - nE(\bar{X}^2)
$$

### 第二步：分别计算 $E(X_i^2)$ 和 $E(\bar{X}^2)$

根据方差与期望的关系：$Var(X) = E(X^2) - [E(X)]^2$，即 $E(X^2) = Var(X) + [E(X)]^2$。

对于单个样本 $X_i$，其总体均值为 $\mu$，总体方差为 $\sigma^2$：

$$
E(X_i^2) = \sigma^2 + \mu^2
$$

对于样本均值 $\bar{X}$，其期望仍为 $\mu$，但方差为总体方差的 $1/n$，即 $Var(\bar{X}) = \frac{\sigma^2}{n}$：

$$
E(\bar{X}^2) = Var(\bar{X}) + [E(\bar{X})]^2 = \frac{\sigma^2}{n} + \mu^2
$$

### 第三步：代入并合并同类项

将第二步的结果代入第一步的公式：

$$
\begin{aligned}
E(S) 
&= \sum_{i=1}^n (\sigma^2 + \mu^2) - n\left(\frac{\sigma^2}{n} + \mu^2\right) \\
&= n\sigma^2 + n\mu^2 - \sigma^2 - n\mu^2 \\
&= (n - 1)\sigma^2
\end{aligned}
$$

### 第四步：对比两种分母的期望值

- **如果除以 $n$（错误做法）**：
$$
  E\left(\frac{S}{n}\right) = \frac{n-1}{n}\sigma^2
$$
  由于 $\frac{n-1}{n} < 1$，该估计值**恒小于**总体方差 $\sigma^2$，属于系统性的**有偏低估**。

- **如果除以 $n-1$（正确做法）**：
$$
  E\left(\frac{S}{n-1}\right) = \frac{(n-1)\sigma^2}{n-1} = \sigma^2
$$

该估计量的数学期望**恰好等于**总体方差，符合统计学上的**无偏性**要求。

### 直观补充自由度视角

计算 $\bar{X}$ 时已经消耗了 1 个约束条件（即 $\sum (X_i - \bar{X}) = 0$），因此 $n$ 个离差中真正能自由变动的独立信息只有 $n-1$ 个。将平方和除以有效的独立信息个数 $n-1$，才是在平均意义上还原总体离散程度的正确操作。
